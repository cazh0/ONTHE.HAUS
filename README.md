Chrome extension that downloads full tracks from lalal.ai by capturing and stitching preview segments.

## Install

1. Download and extract this repo
2. Go to `chrome://extensions/`
3. Enable **Developer mode** (top-right)
4. Click **Load unpacked** → select the extracted folder

## Use

1. Go to lalal.ai and play a preview
2. Click the extension icon
3. Download individual segments or the full track

Segment 000 must be captured before downloading a full track.

## Privacy

Everything runs locally. No data leaves your browser.
